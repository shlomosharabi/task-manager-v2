# 📝 אפליקציית ניהול משימות (Todo PWA)

אפליקציית ניהול משימות מתקדמת עם תמיכה מלאה באופליין, התראות, ועיצוב מודרני בעברית.

## ✨ תכונות

### תכונות עיקריות
- ✅ **ניהול משימות מלא**: הוספה, סימון כהושלמה, ומחיקה
- 📅 **ניהול תאריכים**: משימות משויכות לתאריכים ספציפיים
- ⏰ **התראות חכמות**: הגדרת התראות לשעות ספציפיות
- 💾 **אחסון מקומי**: כל הנתונים נשמרים במכשיר באמצעות IndexedDB
- 📱 **PWA**: התקנה למסך הבית, עבודה אופליין
- 🎨 **עיצוב מודרני**: ממשק נקי ואינטואיטיבי עם אנימציות עדינות
- 🌐 **עברית מלאה**: תמיכה מלאה ב-RTL ועברית

### תכונות נוספות
- בחירת תאריך גמישה (5 שנים אחורה/קדימה)
- סטטיסטיקות בזמן אמת
- פידבקים מונפשים
- עיצוב רספונסיבי (מובייל ודסקטופ)
- תמיכה במצב אופליין מלא

## 🚀 התקנה והרצה

### דרישות מקדימות
- דפדפן מודרני (Chrome, Firefox, Safari, Edge)
- שרת מקומי לבדיקה (אופציונלי אך מומלץ)

### הרצה מקומית

#### אופציה 1: שרת Python (מומלץ)
```bash
# פתח את התיקייה בטרמינל
cd todo-pwa

# הרץ שרת מקומי
python3 -m http.server 8000

# פתח בדפדפן
# http://localhost:8000
```

#### אופציה 2: שרת Node.js
```bash
# התקנת http-server (פעם אחת)
npm install -g http-server

# הרצת השרת
cd todo-pwa
http-server -p 8000

# פתח בדפדפן
# http://localhost:8000
```

#### אופציה 3: Live Server (VS Code)
1. התקן את התוסף "Live Server" ב-VS Code
2. לחץ ימני על `index.html`
3. בחר "Open with Live Server"

### התקנה כ-PWA
1. פתח את האפליקציה בדפדפן
2. לחץ על כפתור "📲 התקן אפליקציה" בראש העמוד
3. אשר את ההתקנה
4. האפליקציה תופיע במסך הבית שלך!

## 📖 הוראות שימוש

### ניהול משימות
1. **הוספת משימה**: הקלד כותרת בשדה העליון ולחץ ➕
2. **סימון כהושלמה**: לחץ על ה-checkbox ליד המשימה
3. **מחיקת משימה**: לחץ על כפתור 🗑️

### ניהול תאריכים
- **מעבר בין ימים**: השתמש בחצים ◀ ▶
- **בחירת תאריך ספציפי**: לחץ על בורר התאריך
- **טווח**: 5 שנים אחורה עד 5 שנים קדימה
- **ברירת מחדל**: תמיד היום הנוכחי

### התראות
1. לחץ על אייקון 🔕 ליד המשימה
2. בחר שעה להתראה
3. לחץ "שמור"
4. בשעה שנבחרה תקבל התראה דחיפה
5. להסרת התראה: לחץ שוב על האייקון ובחר "הסר התראה"

**חשוב**: להתראות צריך לאשר הרשאות בדפדפן בפעם הראשונה.

### סטטיסטיקות
בתחתית המסך תמיד מוצגות:
- סה"כ משימות
- משימות שהושלמו
- משימות בהמתנה

```
│   ├── app.js             # לוגיקה ראשית

├── icons/
│   ├── icon-192.png       # אייקון קטן

```

## 💡 פתרון בעיות נפוצות

### התראות לא עובדות
1. וודא שנתת הרשאות להתראות בדפדפן
2. התקן את האפליקציה כ-PWA (התראות עובדות טוב יותר במצב מותקן)
3. בדוק שהזמן להתראה עדיין לא עבר
4. בדפדפנים מסוימים, התראות עובדות רק כאשר האפליקציה פתוחה

### האפליקציה לא עובדת אופליין
1. וודא שהתחברת לאינטרנט לפחות פעם אחת
2. בדוק את ה-Service Worker בכלי המפתחים (Application → Service Workers)
3. נסה לרענן את העמוד (F5)

### הנתונים נמחקו
- הנתונים נשמרים במכשיר. אם ניקית את המטמון/cookies, הנתונים עלולים להימחק
- השתמש ב-"Private/Incognito Mode" רק לבדיקות, לא לשימוש קבוע

### בעיות תצוגה במובייל
- וודא שה-viewport מוגדר נכון (כבר מוגדר בקוד)
- נסה לסגור/פתוח מחדש את הדפדפן
- התקן כ-PWA לחוויה טובה יותר

## 🎨 התאמה אישית

אם ברצונך להתאים את האפליקציה:

### שינוי צבעים
ערוך את הקובץ `css/style.css` ושנה את המשתנים ב-`:root`:
```css
:root {
    --primary: #6366f1;      /* צבע ראשי */
    --success: #10b981;      /* צבע הצלחה */
    --danger: #ef4444;       /* צבע מחיקה */
    /* ... */
}
```

### שינוי פונט
החלף את קישור ה-Google Fonts ב-`index.html`:
```html
<link href="https://fonts.googleapis.com/css2?family=YOUR_FONT&display=swap" rel="stylesheet">
```

ועדכן ב-CSS:
```css
body {
    font-family: 'YOUR_FONT', sans-serif;
}
```
